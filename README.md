# AI Novel Studio

AI-Native 长篇小说写作平台

## 简介

一个本地优先、AI 增强的结构化写作 IDE，支持无限层级目录、自动实体关联、可视化时间线。

## 技术栈

- **Frontend**: Next.js 15, React, TypeScript
- **Styling**: Tailwind CSS, Shadcn/ui
- **Backend**: Supabase (Auth, PostgreSQL)
- **AI**: Vercel AI SDK

## 开发

```bash
npm install
npm run dev
```
